Sobre o Projeto

Otimização da Distribuição de Redes de Fibra Óptica

Discente

Amanda do Prado Gomes de Carvalho

Distribuição de Redes de Fibra Óptica entre Cidades

Contexto do Projeto

Este projeto aborda o problema de otimização de infraestrutura de telecomunicações, especificamente a distribuição de redes de fibra óptica entre cidades. O objetivo é determinar a melhor rota para instalar cabos de fibra óptica que conectem múltiplas cidades com o menor custo (distância total).

Desafio: Uma operadora de telecomunicações precisa conectar várias cidades através de redes de fibra óptica. Cada conexão entre duas cidades tem um custo (distância, terreno, infraestrutura existente). O objetivo é encontrar a rota ótima que visite todas as cidades exatamente uma vez e retorne ao ponto de partida, minimizando o custo total de instalação.

Formulação Matemática

Este problema é uma instância do Caixeiro Viajante (TSP), onde:

  • Cidades: Pontos de distribuição ou centros urbanos que precisam estar conectados
  • Distâncias (custos): Representam o custo de instalação de fibra óptica entre cada par de cidades
  • Rota: Caminho que instala fibra óptica em todas as cidades com custo mínimo
  • Objetivo: Minimizar a distância total (e consequentemente o custo) de fibra óptica instalada

Métodos Implementados

1. Subida de Encosta (Hill Climbing)

Estratégia que melhora iterativamente a rota. Começa com uma rota aleatória e busca melhorias locais até não encontrar mais. Rápido, mas pode ficar preso em soluções subótimas.

2. Subida de Encosta com Tentativas (Multi-Start)

Executa a Subida de Encosta múltiplas vezes começando de diferentes rotas iniciais. Aumenta a probabilidade de encontrar uma solução melhor ao explorar diferentes regiões do espaço de busca.

3. Têmpera Simulada (Simulated Annealing)

Algoritmo inspirado em processos de resfriamento. Aceita movimentos que pioram a solução com uma probabilidade decrescente, permitindo escapar de soluções locais e explorar mais o espaço de busca.

4. Análise Comparativa

Executa todos os métodos com diferentes configurações de parâmetros e compara os resultados. Permite avaliar qual estratégia oferece melhor desempenho para o problema específico.

Benefícios da Otimização

  • Redução de Custos: Minimizar a distância de fibra óptica reduz significativamente o investimento em infraestrutura
  • Eficiência Operacional: Menos cabos significam menos manutenção e menores custos operacionais
  • Tempo de Implementação: Uma rota otimizada pode ser implementada mais rapidamente
  • Escalabilidade: Os métodos podem ser adaptados para redes com centenas ou milhares de pontos de conexão

Dados de Entrada

O sistema oferece dois tipos de execução:

  • FIXO: Usa um conjunto predefinido de 8 cidades
  • ALEATÓRIO: Gera diferentes cenários de distribuição urbana para testar os algoritmos em variadas situações
